# ⚡ Sai Tele — Developer Portfolio

A responsive, high-performance, dark-themed personal portfolio website built for **Sai Tele**, an AI/ML student and Python developer. Crafted with clean HTML, modular CSS, and vanilla JavaScript, it features interactive particle animations and a fully responsive glassmorphic UI.

🚀 **Live Demo:** [https://saitele325.github.io/portfolio/](https://saitele325.github.io/portfolio/)

---

## ✨ Features

- **Dark Glassmorphic UI** — Space-inspired gradients, glowing borders, and an animated canvas particle background.
- **Scroll & Entrance Animations** — Staggered reveal-on-scroll effects, a hero entrance sequence, animated gradient text, and a smooth page-load transition.
- **Interactive Navbar** — Scroll-aware blur/shadow transitions on desktop and a slide-in mobile drawer menu.
- **Cursor Glow & Progress Bar** — Custom cursor-tracking glow effect paired with a scroll progress indicator.
- **Fully Responsive** — Fluid layouts tuned with dedicated breakpoints for desktop, tablet, and mobile.
- **Featured Sections:**
  - **Hero** — Typing animation effect with direct call-to-action links.
  - **About** — Profile highlights, quick trait cards, and personal background.
  - **Skills** — Categorized display of programming languages, web technologies, and AI/ML tools.
  - **Education** — Structured academic timeline.
  - **Certifications** — Credential cards with PDF preview links.
  - **Projects** — Showcase cards for featured software and web projects with live demo & GitHub repository links.
  - **Resume** — Quick access to a downloadable resume.
  - **Contact** — Floating glassmorphic contact form with animated success state, plus links to email, LinkedIn, and social profiles.

---

## 🛠️ Tech Stack & Dependencies

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Styling & Effects:** Custom CSS variables, glassmorphism, CSS keyframe animations, media-query-driven responsive design
- **Fonts:** [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) & [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- **Icons:** Lightweight inline SVG icons
- **No frameworks, no build step** — pure HTML/CSS/JS, ready to deploy anywhere static files are served

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/saitele325/portfolio.git
   ```
2. Navigate into the project folder:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in your browser, or serve it locally (recommended for correct asset loading):
   ```bash
   npx serve .
   ```

No build tools, package installs, or configuration are required — it's ready to run out of the box.
